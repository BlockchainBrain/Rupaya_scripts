# Rupaya_scripts
	Scripts for the Rupaya Project

#1 Rupaya daemon check and restart script (root users)
	
	Script Name: check-rupaya-root.sh
	
	Install process:
		wget https://raw.githubusercontent.com/BlockchainBrain/Rupaya_scripts/master/check-rupaya-root.sh
		chmod +x check-rupaya-root.sh
		bash check-rupaya-root.sh

#2 Rupaya daemon check and restart script (root users that used the bash script to setup the MN)
